# Three 官网例子

> 基于 vite + vue3 + ts + three 构建 three 官网原生例子，通过例子学习 three API！

THREE使用版本：0.156.0

```bash
# 安装依赖
yarn install

# 本地运行
yarn dev

# 打包
yarn build

# docker 部署 （需要自行登录docker并创建镜像仓库）
./docker/docker_build.sh
```
